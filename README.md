# AI-Builder: Website Intelligence Operating System

Advanced engine that generates professional websites by analyzing internet trends without expensive LLM APIs.

## Features
- **Deterministic Generation**: No AI hallucinations, only clean code.
- **Internet Intelligence Layer**: Real-time analysis of global brands.
- **Persistent Search**: Multi-mirror fallback system.
- **Premium Components**: Tailwind CSS + AOS animations.

## Tech Stack
- **Backend**: Go (Golang)
- **Database**: SQLite3
- **Infrastructure**: Railway + Cloudflare
